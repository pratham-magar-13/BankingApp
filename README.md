BankingApp(Spring Boot)

Features
* Name and balance Enquiry
* Debit and Credit individual user
* Transfer balance from one to another
* Show all users

Tech Stack
* Java 17
* Spring Boot
* Spring Data JPA
* Maven
* MySQL
